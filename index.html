      Help
      Type /help for a list of all commands. Also, support me on Patreon to help me keep the servers running!
      • F1 - Hide part of the GUI
      • G - Toggle grid
      • F - Manually add color
      • Shift + click / Middle click - Move camera
      • Ctrl + Z - Undo pixel (+ Shift = bulk undo)
      • Ctrl + Scroll - Change zoom level
      • P / M / O / L / B - Select Pipette / Move / Cursor / Line / Fill tool
